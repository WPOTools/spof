spof
====

Spof your site.


`sudo npm install spof -g`


usage
----------

in command line run:
`spof --website http://google.com --css --js --port 3001`

- **--js**: SPOF javascripts
- **--css**: SPOF css
- **--port**: sepecify port for the server, default is 3000

**--js and --css are optional but you need to spof either js or css**
